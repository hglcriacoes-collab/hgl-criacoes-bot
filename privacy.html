🔒 Política de Privacidade

Última atualização: 25 de março de 2026

🛡️ Nosso Compromisso: Sua privacidade e segurança de dados são nossas prioridades. Coletamos apenas o mínimo necessário para o funcionamento do serviço.

1. Coleta de Dados

O HGL CRIACOES opera com coleta mínima de dados:

NÃO coletamos:
  • Nome completo
  • Endereço de e-mail
  • Número de telefone
  • Localização geográfica
  • Dados bancários ou de pagamento
Armazenamos localmente (no seu servidor):
  • Credenciais de API (criptografadas)
  • Links de vídeos para processamento
  • Logs de processamento de vídeo
  • Configurações do usuário

2. Como Usamos os Dados

Os dados são utilizados exclusivamente para:

  • Autenticar com a API oficial do TikTok
  • Processar e publicar vídeos automaticamente
  • Manter o funcionamento do bot
  • Diagnosticar problemas técnicos
  • Melhorar o desempenho do sistema

3. Armazenamento e Segurança

Medidas de segurança implementadas:

  • Criptografia de credenciais sensíveis (API keys, tokens)
  • Armazenamento local (dados não são enviados para servidores externos)
  • Repositório GitHub privado (código-fonte protegido)
  • Atualizações regulares de segurança
  • Backup automático de configurações

4. Compartilhamento de Dados

🚫 Não compartilhamos, vendemos ou alugamos seus dados com terceiros, exceto quando estritamente necessário para o funcionamento do serviço:
  • API do TikTok: Apenas para publicação de vídeos (conforme sua autorização)
  • API DeepSeek: Apenas para processamento de IA (quando aplicável)
  • Obrigações legais: Somente se exigido por lei ou ordem judicial

5. Cookies e Rastreamento

O aplicativo não utiliza cookies, pixels de rastreamento ou tecnologias similares para monitorar usuários.

6. Direitos do Usuário (LGPD/GDPR)

Você tem direito a:

  • Acesso: Solicitar cópia dos dados armazenados
  • Correção: Retificar dados incompletos ou inexatos
  • Exclusão: Solicitar eliminação completa dos dados
  • Portabilidade: Exportar suas configurações e dados
  • Revogação: Retirar consentimento a qualquer momento

7. Retenção de Dados

Os dados são mantidos enquanto o aplicativo estiver em uso. Você pode solicitar exclusão total a qualquer momento excluindo os arquivos de configuração.

8. Transferência Internacional

Os dados permanecem armazenados localmente em seu servidor. Não realizamos transferência internacional de dados.

9. Alterações nesta Política

Podemos atualizar esta política periodicamente. Notificaremos sobre mudanças significativas através do repositório GitHub.

10. Conformidade Legal

Esta política está em conformidade com:

  • LGPD (Lei Geral de Proteção de Dados - Brasil - Lei 13.709/2018)
  • GDPR (Regulamento Geral de Proteção de Dados - União Europeia)

11. Menores de Idade

O serviço não é direcionado a menores de 18 anos. Não coletamos intencionalmente dados de crianças.

12. Violação de Dados

Em caso de violação de segurança que afete seus dados, notificaremos dentro de 72 horas e tomaremos medidas imediatas para mitigar danos.
